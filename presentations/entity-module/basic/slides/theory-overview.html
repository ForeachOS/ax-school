Wat is een entity?

An entity is a lightweight persistence domain object. Typically, an entity represents a table in a relational database, and each entity instance corresponds to a row in that table.
persistence (CRUD UI)

Een entity voor EntityModule?

EntityModule laat toe om eender welk object als een entity te configureren.

Door te configureren hoe:

  • Een nieuwe instantie aan te maken
  • De identiteit van een instantie te bepalen
  • Een bestaande instantie op te halen
  • Een instantie te presenteren
  • Een instantie op te slaan of te verwijderen
